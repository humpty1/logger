ПОЯСНИТЕЛЬНАЯ ЗАПИСКА ПРОЕКТА Logger - ЖУРНАЛИРОВАНИЕ МНОГОСРЕДОВЫХ ПРИЛОЖЕНИЙ

Данный документ содержит сведения о назначении динамически подключаемой библиотеки Logger - Журналирование многосредовых приложений. Назначение работы библиотеки заключается в выводе текстовых сообщений различного уровня важности в текстовый файл, находящийся в том же каталоге, откуда запускается приложение и имеющий такое же имя, как и приложение. Файл журнала имеет расширение log. Например, [Тестюнит](@ref test.Program)-а приложение имеет название _app.exe_, а журнал _app.log_. В случае, если приложение консольное, есть возможность дублирования некоторых сообщения в стандартный вывод ошибок.
Написан еще один юнит-тест, который показывает динамическую загрузку библиотеки Logger (то есть, на шагы выполнения, без указания линкеру использовать эту сборку).
