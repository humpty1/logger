ПОЯСНИТЕЛЬНАЯ ЗАПИСКА ПРОЕКТА Logger - ЖУРНАЛИРОВАНИЕ МНОГОСРЕДОВЫХ ПРИЛОЖЕНИЙ

Данный документ содержит сведения о назначении динамически подключаемой библиотеки Logger - Журналирование многосредовых приложений. Документ является Пояснительной записка к проекту \nm и был создан утилитами Doxygen (см. http://doxygen.org) и Microsoft's HTML Help Workshop (см.http://www.microsoft.com/en-us/download/details.aspx?id=21138). С утилитой Doxygen можно познакомится в документах DOXYGEN И GRAPHVIZ: ДОКУМЕНТИРОВАНИЕ ПРОЕКТОВ НА C# (см. http://agp1.hx0.ru/dgIntro.html), БИБЛИОТЕКА РАЗБОРА АРГУМЕНТОВ КОМАНДНОЙ СТРОКИ (C#, Doxygen и Microsoft's HTML Help Workshop) (см. http://agp1.hx0.ru/args.html), Динамически подключаемая библиотека Logger написана на языке C# c использованием .Net версии 4.0. При разработке тестюнита для тестирования библиотеки использовалась Библиотека разбора аргументов командной строки (см. http://agp1.hx0.ru/args.html) Полная поставка представляет собой архив Logger.X.XX.zip (где X.XX версия библиотеки), который можно взять по адресу http://agp1.hx0.ru/Logger.html. Назначение работы библиотеки заключается в выводе текстовых сообщений различного уровня важности в текстовый файл, находящийся в том же каталоге, откуда запускается приложение и имеющий такое же имя, как и приложение. Файл журнала имеет расширение log. Например, [Тестюнит](@ref test.Program)-а приложение имеет название _app.exe_, а журнал _app.log_. В случае если приложение консольное, есть возможность дублировать некоторые сообщения в стандартный вывод ошибок.
